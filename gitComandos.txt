Repositorios:

git init -> crear repositorio local nuevo
git clone -> clonar un repositorio existente (copia)

git add . -> Agregar o guardar cambios
git commit -m "Commit message" -> Actualizaciones del programa con mensajes cortos para saber que fue
git push origin main -> Enviar los cambios
git pull origin main -> Traer los cambios

git remote add origin <server> -> Clonarlo en la nube

git merge nombre_rama2 -> Traer y juntar los cambios de otra rama
